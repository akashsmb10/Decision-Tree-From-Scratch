# 🌳 Decision Tree Implementation

This project demonstrates the implementation of a **Decision Tree algorithm using scikit-learn** in Python. It covers core machine learning concepts like entropy, information gain, and tree building.

---

## 📌 Features

- Decision Tree using scikit-learn
- Tree visualization
- Model training and prediction
- Performance evaluation
- Hyperparameter tuning

---

## 🧠 Concepts Covered

- Entropy
- Information Gain
- Overfitting & Depth Control

---

## 🛠️ Tech Stack

- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## 📂 Dataset

- Iris dataset from scikit-learn
- 150 samples, 4 features, 3 classes

---

## 📊 Results

- Accuracy: ~95% (with tuned parameters)
- Observations:
  - Model performs well on the test set
  - Overfitting observed with deeper trees

---

## ▶️ How to Run

```bash
git clone https://github.com/akashsmb10/Decision-Tree-From-Scratch.git
cd Decision-Tree-From-Scratch
pip install -r requirements.txt
jupyter notebook
```
